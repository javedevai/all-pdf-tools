## üí° Pro Tips

### Repair PDF
- Start with "standard" mode
- Use "aggressive" only for severely damaged files
- Keep corrupted pages as placeholders to see what's missing

### OCR PDF
- Select correct language for best results
- Enable deskew for scanned documents
- Enable enhance for low-quality scans

### Compare PDF
- Use "text" mode for contract reviews
- Use "visual" mode for design changes
- Use "both" for comprehensive analysis

### Optimize for Web
- Use "medium" compression for balanced results
- Keep fonts embedded for consistent display
- Remove metadata for privacy

### Edit Metadata
- Add keywords for better searchability
- Set proper title for document management
- Include author for attribution

### Viewer Preferences
- Use "FullScreen" for presentations
- Use "UseThumbs" for navigation
- Set "TwoColumn" for book-like reading

### Extract Fonts
- Use for design consistency checks
- Verify font licensing before distribution
- Identify fonts for replacement

### Analyze PDF
- Use for troubleshooting PDF issues
- Verify PDF specifications
- Quality assurance testing

### Batch Process
- Process similar files together
- Use "compress" for file size reduction
- Use "watermark" for branding

### Print Ready
- Always use CMYK for commercial printing
- Add 3-5pt bleed for full-bleed designs
- Enable crop marks for trimming guides

### Share Link
- Use 30 minutes for quick shares
- Enable password for sensitive documents
- Disable download for view-only access

## üêõ Troubleshooting

### Tool Not Working?
1. Check file is valid PDF
2. Try smaller file first
3. Check browser console for errors
4. Clear browser cache
5. Try different browser

### Processing Too Slow?
1. Reduce file size first
2. Use lower quality settings
3. Process fewer files at once
4. Close other browser tabs
5. Use desktop browser (not mobile)

### Output Not as Expected?
1. Review option settings
2. Try different mode/quality
3. Check input file quality
4. Verify file isn't corrupted
5. Test with sample file first

### Browser Crashes?
1. File may be too large
2. Reduce quality settings
3. Process in smaller batches
4. Close other applications
5. Use browser with more memory
